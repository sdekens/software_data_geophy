This is a README file for the installation of PyGimli.

1. Execute the script "install_anaconda.sh" by running the following command: bash install_anaconda.sh
    - If the scripts asks your permission to download some packages, type y or o depeding on what's prompted
    - The script will prompt several questions for the installation of Anaconda, always press ENTER
    - This may take several minutes

2. Execute the script "install_pygimli.sh" by running the following command: bash install_pygimli.sh

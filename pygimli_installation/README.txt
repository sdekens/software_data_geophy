This is a README file for the installation of PyGimli.

1. Download the Anaconda installer via this link : https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh

2. Place the installer in this current directory

3. Execute the script "install_miniconda.sh" by running the following command: bash install_miniconda.sh
    - If the scripts asks your permission to download some packages, type y or o depeding on what's prompted
    - The script will prompt several questions for the installation of Anaconda, always press ENTER
    - This may take several minutes

4. Execute the script "install_pygimli.sh" by running the following command: bash install_pygimli.sh

5. Once it is done, you can activate and deactivate your conda environment by running the command : conda activate pg OR conda deactivate
